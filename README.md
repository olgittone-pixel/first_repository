# first_repository
задача 3.14 У вас есть список ежемесячных продаж товаров в течение года. Напишите программу, которая сначала запрашивает количество товаров, а затем для каждого из них запрашивает строку, содержащую количество продаж за каждый из 12 месяцев. После получения всех данных, программа должна вычислить суммарные продажи для каждого месяца и найти месяц с максимальной и минимальной суммарной продажей. Выведите номера месяцев на экран каждый в отдельной строке.
